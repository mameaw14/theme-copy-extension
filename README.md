# Theme Copy Extension

## Install dependencies

`npm i`

## Build

`npm run build`

## Build Result

Result of building will be in `/src/build`  
There are 2 files `applyTheme.js` and `buildTheme.js`  
These two files will be called from `/src/popup.js`

## Extension

Final result

- /images/\*/
- /src
  - /build
    - applyTheme.js
    - buildTheme.js
  - background.html
  - background.js
  - popup.html
  - popup.js
- manifest.json
